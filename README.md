# Portfólio - William Chapuis

Portfólio pessoal hospedado gratuitamente no GitHub Pages.

## Como publicar
1. Commitar os arquivos (`index.html`, `styles.css`, `script.js`) na branch `main`
2. Settings → Pages → Deploy from a branch → `main` / root
3. Aguardar o link do GitHub Pages aparecer

## Como editar projetos
Abra `script.js` e edite o array `projects`.
